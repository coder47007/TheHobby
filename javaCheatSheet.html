JavaScript Cheat Sheet

This cheat sheet is a guide to the basic and essential JavaScript syntax and concepts for beginners.

Variables

In JavaScript, we can store data in variables using var, let, or const.


let name = "John";   // using let (can be reassigned)
const age = 25;      // using const (cannot be reassigned)
var city = "New York"; // using var (older approach)

Example:


console.log(name); // Output: John
console.log(age);  // Output: 25

Data Types

JavaScript has several data types including strings, numbers, booleans, arrays, objects, etc.


let string = "Hello World";   // String
let number = 42;              // Number
let isActive = true;          // Boolean
let colors = ["Red", "Green", "Blue"]; // Array
let person = { name: "John", age: 30 }; // Object

Functions

A function is a block of code designed to perform a particular task.


function greet(name) {
    return "Hello, " + name;
}

console.log(greet("John")); // Output: Hello, John

Conditionals

Conditionals are used to perform different actions based on different conditions. We use if, else, and else if.


let number = 10;

if (number > 5) {
    console.log("Greater than 5");
} else {
    console.log("Less than or equal to 5");
}

Output: Greater than 5

Loops

Loops are used to execute the same code repeatedly until a condition is met. Common loops include for and while.


// For loop
for (let i = 0; i < 5; i++) {
    console.log(i); // Output: 0 1 2 3 4
}

// While loop
let count = 0;
while (count < 5) {
    console.log(count); // Output: 0 1 2 3 4
    count++;
}

Arrays

An array is a special variable that can hold multiple values at once.


let fruits = ["Apple", "Banana", "Cherry"];

console.log(fruits[0]); // Output: Apple

fruits.push("Orange");   // Adds Orange to the array
console.log(fruits); // Output: ["Apple", "Banana", "Cherry", "Orange"]

Objects

Objects are collections of key-value pairs and are used to store related data.


let person = {
    name: "John",
    age: 30,
    greet: function() {
        return "Hello " + this.name;
    }
};

console.log(person.name);  // Output: John
console.log(person.greet()); // Output: Hello John

Events

Events are actions that occur in the browser, such as a user clicking a button or pressing a key. You can add event listeners to elements to handle events.


document.getElementById("button").addEventListener("click", function() {
    alert("Button clicked!");
});

DOM Manipulation

The DOM (Document Object Model) represents the structure of an HTML document. JavaScript can be used to manipulate it, changing content, structure, or style.


// Changing text content
document.getElementById("demo").textContent = "New Text!";

// Changing styles
document.getElementById("demo").style.color = "blue";

This is a demo text.

Debugging

Use console.log() to display values in the console for debugging.


let x = 10;
console.log(x); // Output: 10
